# Changelog

## v1.2

### Added

- Добавлен модуль `modules/20_install_3xui.sh`.
- Добавлена идемпотентная проверка установленного 3x-ui.
- Добавлена проверка и установка базовых зависимостей для 3x-ui.
- Добавлена проверка службы `x-ui` после установки.
- Добавлен вывод IP сервера и подсказок по доступу к панели.
- Добавлены базовые документы `CHANGELOG.md` и `ROADMAP.md`.

### Changed

- `install.sh` обновлен до версии v1.2.
- Пункт меню `Установить 3x-ui` теперь вызывает отдельный модуль.
- Главное меню теперь возвращается после выполнения выбранного действия.
- README обновлен под новую модульную архитектуру.

### Not included

- SSL, Nginx, сертификаты и DNS не входят в v1.2 и запланированы на v1.3.
- `PROJECT_HISTORY.md` не создавался в GitHub, потому что актуальная версия файла находится локально и должна быть синхронизирована отдельно.
