# BlackSwan Bootstrap

Минимальный стартовый набор для быстрой проверки нового VPS перед установкой VPN-панели.

## Главная задача

1. Проверить, открывается ли сервер по HTTP.
2. Измерить примерную скорость соединения между устройством пользователя и сервером.
3. Если результат нормальный, установить 3x-ui отдельной командой.
4. Запускать проверку на любом новом сервере из GitHub.
