# Project History

## BlackSwan Bootstrap

BlackSwan Bootstrap создается как долгосрочная модульная система подготовки Ubuntu VPS под инфраструктурные проекты BlackSwan Lab.

Главная идея проекта: чистый VPS должен быстро превращаться в готовую рабочую платформу для VPN, Telegram-ботов, Python-проектов, Docker, сайтов, API, AI-агентов и будущих сервисов.

## Основные инженерные правила

- Один модуль отвечает за одну задачу.
- Главное меню только вызывает модули.
- Логика установки хранится внутри модулей.
- Повторный запуск должен быть безопасным.
- Если компонент уже установлен, модуль сообщает об этом и не ломает сервер.
- Каждая версия должна улучшать архитектуру и документацию.
